数据库连接池

数据库连接池是一个容器，用来分配、释放数据库连接。

连接的建立和释放都是需要占据资源的，因此在一个容器内预先创建好一部分连接资源，在进行动态的管理和分配，降低了资源的消耗，降低了用户的一个使用时间。

数据库连接池通常是由第三方来实现的，因此以后不需要再通过Manager来创建Connection对象。
